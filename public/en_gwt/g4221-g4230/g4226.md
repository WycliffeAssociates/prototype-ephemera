# ποῦ/pou
This word is used to ask something about an action or situation. It is an adverb.
* This word can mean where. It is used to ask a question.
    * It can be used to ask where someone or something is.
    * It can be used to ask where something happens.
    * It can be used to ask where someone goes.
* This word can mean under what circumstances or in what situation.
