# καλός/kalos
This word can describe someone or something. It is an adjective.

* This word describes someone or something good.
    * This word can be used to describe someone or something that is beautiful.
    * This word can be used to describe something that is good for people to use for a particular purpose.
    * This word can be used to describe someone or something that is right or fair.
    * This word can be used to describe someone or something that is noble or honorable.
    * This word can be used to describe one particular thing that is better than another thing.

Sometimes this word is used with other words:

* When this word is used with λιμήν (G3040), these words mean Fair Havens. This is the name of a harbor.
