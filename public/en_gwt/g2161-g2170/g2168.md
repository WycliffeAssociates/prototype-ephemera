# εὐχαριστέω/eucharisteō
Sometimes this word is an action. It is a verb.

* This word can mean to thank or give thanks. When used in this way, this word is used to talk about a person thanking some other person.

Sometimes this word describes a situation or how something is. It is a verb. However, it is not an action. 

* This word can mean to be thankful. When used in this way, this word is used to talk about a person having a thankful attitude.

**Advice to translators**: This type of verb describes a state of being.
